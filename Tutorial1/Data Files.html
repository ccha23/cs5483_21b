Create an ARFF fileÂ¶

Create an ARFF file named AND.arff in the current directory for the AND gate \(Y=X_1\cdot X_2\). Use 0 and 1 to represent False and True respectively.

[See the documentation for the ARFF file format, or take a look at some of the ARFF files in the WEKA data folder as examples.]

Hint: To generate the file from this notebook directly:

  1. Copy the solution template below to the following solution cell.

  2. Replace each underscore _ by an appropriate value.

  3. Execute the code cell.

content = '''
@RELATION AND
@ATTRIBUTE X1 {0, 1}
@ATTRIBUTE X2 {_, _}
@ATTRIBUTE Y {_, _}
@DATA
_, _, _
_, _, _
_, _, _
_, _, _
'''
